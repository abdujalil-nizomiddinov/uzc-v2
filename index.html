UZCODE(uzc) Tili Haqida

UZCODE(uzc) - bu yangi boshlovchilar uchun mo'ljallangan, JavaScript asosidagi dasturlash tili. U o'zining oddiy va tushunarli sintaksisi, o'zbek tilidagi izohlari va qulay imkoniyatlari bilan ajralib turadi.

Asosiy Buyruqlar

  • print(...) - Ma'lumotni konsolga chiqarish
  • let - O'zgaruvchi e'lon qilish: let x = 5
  • if / elif / else - Shartli operatorlar
  • function - Funksiya yaratish: function nomi(param) { ... }
  • type(...) - O'zgaruvchi turini aniqlash: type(ozgaruvchi)
  • list - Ro'yxat yaratish: let royxat = [1, 2, 3]
  • {...} - Matn ichida o'zgaruvchini chiqarish: "Salom, {ism}"
  • # - Izoh yozish uchun: # Bu izoh

Misollar

# Konsolga chiqarish
print("Salom, Dasturchi!")
# O'zgaruvchilar bilan ishlash
let a = 10
let b = 5
print("Yig'indi:", a + b)
# Ro'yxat bilan ishlash
let mevalar = ["Olma", "Banan", "Anor"]
print("Mevalar:", mevalar)
# Shartli operatorlar
let yosh = 20
if yosh >= 18 {
    print("Siz balog'at yoshidasiz")
} elif yosh >= 13 {
    print("Siz o'smirsiz")
} else {
    print("Siz bola ekansiz")
}
# Funksiya yaratish
function salomBer(ism) {
    print("Salom, {ism}")
}
salomBer("Jasur")
# Typeni aniqlash
let son = 5
print("Turi:", type(son))
# Matnli o'zgaruvchi bilan
let ism = "Ali"
print("Salom, {ism}")
